SVG Graphics


The SVG graphic format first began to be developed in 1999. [3]  It has been adopted as a standard by the World Wide Web Consortium ( W3C ). SVG images can be resized without losing any of the image quality. The technology is mature and allows the Web developer to create custom 2D images for their website.
