Early SVG Animation


The SVG graphic format first began to be developed in 1999. [9]  It has been adopted as a standard by the World Wide Web Consortium ( W3C ). SVG images can be resized without losing any of the image quality. The technology is mature and allows the Web developer to create custom 2D images for their website. It's possible to animate an SVG. One way to do so is by using JavaScript to manipulate the DOM. These are early attempts at animation using SVG. They run inside a web browser, and were written in 2013.
