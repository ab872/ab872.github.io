AngularJS shopping cart demo


AngularJS is a free open source front-end framework. It was originally developed around 2008 or 2009, and is now maintained by Google. [40] [70]

It was an evolutionary improvement over the old way of coding web applications. It added extensions to the existing APIs used for Web development. AngularJs is mostly used for single-page web applications (SPAs), and uses dynamic HTML instead of static HTML. This allows the application to be more responsive to user actions. The design follows MVC (Model, View, Control). [40] [70]

AngularJS is licensed under the Apache license version 2.0
    https://www.apache.org/licenses/LICENSE-2.0

It is used by thousands of developers world-wide. [40] [70] Support for AngularJS officially ended in January 2022 [29], but it is still possible to get extended support for the framework until 2030. [36]

About this demo

This is a simplified version of a shopping cart. It illustrates basic data manipulation in AngularJS.

The application generates an invoice containing the customer information, and the order items. It calculates and displays the sub-total, the tax and the total price for the order.

The customer database is built in JSON and contains three entries. If you click the ? button you can see the customer membership numbers.

If you enter one of the existing membership numbers, the app will automatically populate the invoice with the customer information.

You can order pants and shirts of various sizes and colors. After you select the quantity, type of clothing, the color and the size you can click the Add to Order button to add the item to the order. When you do so, the sub-total, the tax and the total price for the order will be updated and displayed.
