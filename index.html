A collection of various projects and experiments.



CAD/CAM 


Gerber file viewer
File viewer which parses
Gerber files in order to display
a preview image of how a
printed circuit board would look
when manufactured. The image
is rendered using SVG graphics.
Includes references.




Database 


CRUD databases - part 1, arrays
A demonstration of CRUD database programming based on arrays. CRUD stands for Create, Read, Update and Delete. These are the four basic operations every database must perform. Includes references.


CRUD databases - part 1, arrays
Documentation for CRUD databases - part 1, arrays. Includes references.


CRUD databases - part 2, JSON
Documentation for CRUD databases - part 2, JSON. Includes references.


CRUD databases - part 3, JSON
Documentation for CRUD databases - part 3, Traveler's Database using JSON. Includes references.




Graphics


SVG Graphics
A short discussion
of SVG graphics with
references and examples.


Early SVG Animation
It's possible to animate an SVG.
One way to do so is by using
JavaScript to manipulate the DOM.
These are early attempts at animation
using SVG. They run inside a web
browser, and were written in 2013. A
short discussion of SVG animation
with references and examples.
Includes a walkthrough of the
code for a simple animation.


Ray Tracing Graphics
A short discussion
of ray tracing graphics
with references and examples.




Miscellaneous 


Photo Gallery
A collection of photos arranged in a
grid pattern using CSS GRID.
Includes references.

Color Picker
Color picker which allows
visual comparison of two colors.
Gives a visual check of the
contrast between the two.
Zoom sample text up or down in
size to check its appearance.
Includes references.
