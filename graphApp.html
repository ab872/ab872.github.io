Using the Canvas in C#



Operating System:               Windows 7 - 10              
Development IDE:               Visual Studio                 
Language:                               C#                                    
Demo features:                                                                                               
• 1. Drawing on the Windows graphic Canvas.                                                                                           
• 2. Data inspection via zoom in or zoom out.                                                                                               
• 3. Dynamic repositioning of the viewing window.                                                                  
• 4. Selectable data cursors.                                                                                                                                        
• 5. Step through the data using the cursor.                                                      

This is a learning experience focusing on drawing graphics on the screen in C#. It was written in 2010. The application graphs sets of data. Up to ten graphs may be displayed on screen simultaneously. The user may zoom in or zoom out on the data for closer inspection, and may reposition the viewing window if the data falls outside the data display area. The user may step through the data using a selectable cursor.
