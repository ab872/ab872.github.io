Vue.js shopping cart demo


Vue.js was created in 2014 by Evan You. It has over 1.5 million users worldwide, and is downloaded close to 10 million times a month on npm. Some of the companies which use Vue.js are Wikimedia Foundation, NASA, Apple, Google, Microsoft, GitLab, Zoom, Tencent, Weibo, Bilibili, and Kuaishou. The latest version of Vue.js is called Vue 3. It contains new features which make it incompatible with the previous version, Vue 2.7. Vue 2 reached its End of Life on 31 December 2023. [31]

Vue.js has adopted 2 API styles, the Options API and the Composition API. The Options API more closely follows the class model seen in languages which implement Object Oriented Programming ( OOP ). The Composition API has a more flexible layout based on reactive state. [30]

Vue.js is licensed under the MIT license.

About this demo

This is a simplified version of a shopping cart. It illustrates basic data manipulation in Vue.js. The application is written using the Options API in Vue.js.

The application generates an invoice containing the customer information and the order items. It calculates and displays the sub-total, the tax and the total price for the order.

The customer database is built in JSON and contains three entries. If you click the ? button you can see the customer membership numbers. If you enter one of the existing membership numbers, the app will automatically populate the invoice with the customer and the tax information.

You can order pants and shirts of various sizes and colors. When you select the type of clothing, the price is automatically populated from the JSON product database. After you select the quantity, type of clothing, the color and the size you can click the Add to Order button to add the item to the order. When you do so, the sub-total, the tax and the total price for the order will be updated and displayed.
