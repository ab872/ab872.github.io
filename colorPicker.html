Color Picker



Basic text and color manipulations using the DOM. HTML and CSS have several predefined colors. One may refer to a color using its predefined name, or in the case of custom colors with no predefined name, one may specify the Red, Green and Blue ( RGB ) components of the color.

This demo can actually be useful. It allows two colors to be compared to see if they look good next to each other. It also aids in determining if there is enough contrast between the two colors to help comply with the Web Content Accessibility Guidelines ( WCAG ). It also allows one to scale some sample text up or down in size to give a visual indication of how that font size would look on a particular device.


Note:     Depending upon the browser in use, it may be necessary to set the Browser Zoom setting
at around 50% in order to operate the demo without having to scroll up and down.
